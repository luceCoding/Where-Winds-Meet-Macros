# Poison farm script
Meant to farm Aureate Pavilon.

# Poison farm setup
1. Edit the config.yaml to match your keyboard controls.
2. Follow the instructions in this [link](https://youtu.be/gEUdQH8jEQE?si=OjqZXOR5-kkXBSf-) to setup the farm.
3. Make sure you have the panacea fan out and using the meteor flight lightness skill.
4. Face the corner of your wall tower.
5. Open command prompt as Admin.
6. cd "your/script/folder/path"
7. python start_poison_farm.py
8. Use Ctrl + C in the command prompt or the dedicated key in your config.yaml to stop the program.
