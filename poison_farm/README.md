# Poison farm script
Meant to farm Aureate Pavilon.

# Poison farm setup
1. Follow the instructions in this [link](https://youtu.be/gEUdQH8jEQE?si=OjqZXOR5-kkXBSf-) to setup the farm.
2. Make sure you have the panacea fan out.
3. Face the corner wall tower.
4. Open command prompt as Admin.
5. cd "your/script/folder/path"
6. python start_poison_farm.py
7. Use Ctrl + C to stop the program in the command prompt.
